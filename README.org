

* Usage

** Install

#+BEGIN_SRC sh
cd <project-path>
FORMULARS_PATH=`pwd`
sudo ln -s "$FORMULARS_PATH/file_root" /srv/salt
sudo ln -s "$FORMULARS_PATH/pillar_root" /srv/pillar
#+END_SRC

打开 /etc/salt/minion，设置为本地模式:
#+BEGIN_SRC conf
file_client: local
#+END_SRC


** Run state:

#+BEGIN_SRC sh
sudo salt-call --local -l debug state.sls salt.minion
#+END_SRC
